# Java Practice Repository

This workspace contains Java practice projects and examples focused on Object-Oriented Programming, core Java concepts, Spring Boot, Spring Framework, JDBC/Servlets/JSP, and interview preparation.

## Contents

- `AccessModifiers/` – examples for access control and encapsulation
- `Array/` – array operations and sample programs
- `BasicConcepts/` – fundamental Java concepts and small demos
- `CoreJava/` – extensive core Java topics and learning materials
- `GraphQL/` – GraphQL learning examples
- `HackerRank/` – coding challenge solutions
- `Inheritance/` – inheritance and polymorphism examples
- `Interview-Dec25/` – interview practice problems and algorithms
- `JDBCServletsJSPs/` – JDBC, Servlets, JSP, and web app examples
- `MyProjects/` – personal Java projects such as library management
- `Playground/` – sandbox projects and experiments
- `reactive/` – reactive programming and Gradle-based project setup
- `SpringBoot/` – Spring Boot sample applications and exercises
- `SpringFramework/` – Spring Framework tutorials and examples

## Purpose

This repository is used for Java learning, practice, and reference. It includes example code, exercises, and small applications for improving Java skills and exploring related technologies.
